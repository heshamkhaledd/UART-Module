# UART-Module
Implemented a customizable UART peripheral module that supports different configurations:<br/>
<br/>
• Varying Data bits (5 to 9 bits) with the default of 8 bits<br/>
• Even or Odd Parity bit <br/>
• One or Two Stop bits <br/>
